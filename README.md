# 🌍 GlobalPath Education – Study Abroad Consultancy Website

A modern **React + TypeScript + Vite** landing page built for a **study abroad education consultancy**.
The website highlights services like **university admissions guidance, visa assistance, and study abroad counseling**.

This project was built mainly for **learning modern frontend development** while creating something that resembles a **real consultancy website**.

* Modern **hero landing section**
* **Service cards** for consultancy offerings
* Clear **call-to-action buttons**
* **Dark UI design**
* **Responsive layout**
* Built with **React + TypeScript + Vite**

## 📦 Installation

Clone the repository:

```bash
git clone <your-repo-url>
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
